# PEG for edu

## 文法

### 規則

```
N <- e
```

### 文字列リテラル `""`

```
"literal"
```

### 文字クラス `[]`

```
[a-z]
```

### ワイルドカード `.`

```
.
```

### 連接 `e1 e2`

```
e1 e2
```

### 選択 `e1 / e2`

### 繰り返し `e*`

### 1回以上の繰り返し `e+`

### 0回または1回 `e?`

### 肯定先読み `&e`

### 否定先読み `!e`

### 非終端記号 `N`

### コメント

`--` から行末までをコメントとして扱う。

```
-- コメント
```

## 例

```
expr <-  space
       / term ( "+" term
              / "-" term
              )

term <- factor ( "*" factor
               / "/" factor
               / "%" factor
               )

factor <- "(" expr ")" / number

number <- digit+

digit <- [0-9]
```
